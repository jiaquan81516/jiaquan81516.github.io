Final Year Project

Datalogger for BlueSG

This project was in collaboration with BlueSG, a Web Application was created to display the real time data of the Lithium Ion Battery every second from the datalogger which was also self created in PCB form and coded in C for the hardware. The backend server was created using Firebase. This project allowed me to gain more software skills and expose to industry projects. As the Leader of the 3 men team, I allocated the tasks according to what they are stronger at but not disregarding our learning process as at the end of the day, I made sure each of us understood one another's task up to an extent.

Some key technologies that were used are HTML5, CSS3 and JavaScript for the Web App, Fusion360 for the 3D Modelling, and EasyEDA for PCB creation. Here are some pictures of the progress and a video of the final product!
