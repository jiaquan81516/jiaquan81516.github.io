Blood Pressure Monitor

Embedded Systems Programming

This was a Microcontroller project done in year 2. A blood pressure monitor which includes display, alert, display records, low-power mode. Using C++, My group made use of ADC, Interrupts, PWM, and Timers. Personally, this was the toughest module that I have faced mainly because I am not as proficient in C++ and it took me really long to understand the hardware commands as well. Thankfully, with the support of my peers and the lecturer, this project was made possible.
